# Contributing to Claude Code Art Gallery

Thank you for your interest in contributing to the Claude Code Art Gallery! We love seeing creative ASCII art from the community.

## How to Contribute

### 1. Create Your ASCII Art

Create your ASCII artwork in a `.txt` file. Here are some guidelines:

- **Dimensions**: Keep it reasonable (typically 60-80 characters wide, 15-25 lines tall)
- **Characters**: Use standard ASCII characters that display well in terminals
- **Theme**: Claude Code, programming, AI, or creative tech themes work great!
- **Creativity**: Be creative! Minimalist, retro, neon, cosmic - all styles welcome

### 2. Test Your Artwork

Before submitting, test your artwork:

```bash
# Preview in terminal
cat your-artwork.txt

# Test with Claude Code (optional)
./install.sh your-artwork
```

### 3. Submit Your Artwork

**Option A: Pull Request (Recommended)**

1. Fork this repository
2. Add your artwork file to `artwork/community-submissions/your-name-theme.txt`
3. Name your file descriptively: `username-theme.txt` (e.g., `sarah-cyberpunk.txt`)
4. Create a pull request with:
   - Clear title: "Add [Your Theme] artwork"
   - Description of your artwork
   - Screenshot or preview in the PR description

**Option B: Issue Submission**

1. Create a new issue with the "Artwork Submission" label
2. Paste your ASCII art in a code block
3. Describe your artwork and inspiration
4. We'll add it for you!

## Review Process

1. **Submission**: You submit via PR or issue
2. **Review**: Maintainers check for guidelines compliance
3. **Feedback**: We may suggest small tweaks
4. **Merge**: Once approved, your art joins the gallery!
5. **Credit**: You'll be added to our contributors list

## Code of Conduct

Be respectful, creative, and have fun! We're building a positive community of ASCII artists.

## Questions?

- Open an issue with the "question" label
- Tag @yksanjo for maintainer attention
- Check existing artwork for inspiration

## Recognition

All contributors will be:
- Listed in the README credits section
- Mentioned in release notes when their art is added
- Part of a growing community of creative developers!
